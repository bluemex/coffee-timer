# 手沖職人指南：四六沖法 + 花朵繞法 (Coffee Timer)

[![Live Demo](https://img.shields.io/badge/Website-Live%20Demo-blue?style=for-the-badge&logo=coffeescript)](https://coffee-timer.yooneko.com/)

這是一個專為咖啡愛好者設計的網頁版手沖計時器。結合了著名的 **Tetsu Kasuya 4:6 沖煮法** 與獨特的 **花朵繞法 (Flower Pouring)** 技術，透過視覺化的動態圖表與分段計時，引導使用者精準完成每一階段的萃取。

## 🌟 功能亮點

- **4:6 沖煮法邏輯：** 自動將沖煮過程分為五個階段，每段 45 秒。
  - 前兩段 (40%)：調整酸甜平衡。
  - 後三段 (60%)：調整濃度與厚實度。
- **動態視覺引導：** 每個階段配有專屬的 SVG 動態圖解（如中心注水、花瓣式繞圈、對流繞法）。
- **內建快速預設：** 提供兩組標準參數：
  - **300ml：** 使用 20g 咖啡粉。
  - **600ml：** 使用 35g 咖啡粉。
- **智慧自動捲動：** 隨著計時進行，畫面會自動平滑捲動至當前進行中的沖煮階段。
- **響應式設計：** 完美支援手機與桌面瀏覽器。
- **職人語錄：** 沖煮過程中會顯示激勵語錄，陪伴你完成一杯完美的咖啡。

## 🛠️ 使用技術

本專案採用單一檔案架構，不需要複雜的建置步驟，直接開啟即可執行：

- **核心框架：** React 18 (透過 CDN 載入)
- **UI 樣式：** Tailwind CSS (透過 CDN 載入)
- **腳本解析：** Babel Standalone
- **圖示與動畫：** 原生 SVG 與 CSS Keyframes 動畫

## ☕ 沖煮法說明

本計時器預設將總水量分為五等分：

1. **第一段 (0:00 - 0:45)：** 從中心向外繞一圈，定調風味（可依喜好調整水量來強化酸感或甜感）。
2. **第二段 (0:45 - 1:30)：** 中心注水，建立咖啡主體。
3. **第三段 (1:30 - 2:15)：** 花瓣式注水，增加風味廣度。
4. **第四段 (2:15 - 3:00)：** 重複花瓣式注水，提取厚實口感。
5. **第五段 (3:00 - 3:45)：** 大水流帶動對流，平衡最終濃度。
